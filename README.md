# blogger ‚úç

A cross-platform mobile blog app using Flutter. It's a basic CRUD(create, read, update and delete) application with Firebase as the backend database. It allows authenticated users to use all CRUD features.

## Features

* Registering users with email and password
* Logging in registered users
* Logging out signed-in users
* Reading from and Writing to the database
* Updating and Deleting data from the database

## Dependencies

* image_picker
* firebase_auth
* firebase_database
* firebase_storage
